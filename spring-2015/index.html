Creative Coding for the Web

Welcome to IML 400 - Fall 2014!

IML400 is a practical foundational course on creative web design, development and analysis. It provides a conceptual framework to understand, analyze, and design for the web of today, and it serves as a practical introduction to HTML, CSS and Javascript as building blocks.
